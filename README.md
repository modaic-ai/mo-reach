# mo-reach

mo-reach is a collection of GTM tools powered by Modaic. The repo is organized
as small, focused tools that use Modaic to evaluate, route, and improve outreach
workflows.

## Tools

| Tool | Description |
|---|---|
| [mo_buzz](./mo_buzz/) | Reddit monitoring for GTM signals. It scans target subreddits, judges posts with a Modaic Arbiter, applies a confidence-based surfacing policy, and sends actionable items to Slack for review and annotation. |
